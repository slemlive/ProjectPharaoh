2015년 당시 19살때 제작한 프로젝트 입니다.

3D 공부 목적으로 DirectX 9 Samples 중 Obj Parser 를 참고하여 제작 하였습니다.

기본적으로 ASE 파일을 파싱하도록 구성되어 있으며 DirextShow를 통한 영상 재생 기능도 추가되어 있습니다.

사용된 기술 스택
- C++
- DirectX 9
- DXUT

Framework의 기본 기능 목록
- Sprite & Animation
- UI Font
- Sound
- 3D Model & Animation ( Not SkinnedMesh )
- Parse ASE
- DirectShow
